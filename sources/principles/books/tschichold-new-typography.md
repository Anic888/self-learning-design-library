# Tschichold — The New Typography and the Later Classical Return

*Jan Tschichold, "Die neue Typographie" (1928) and "The Form of the Book" (1975). Two phases by the same master — LLM-actionable rules for knowing which to invoke.*

## The two-phase framework — choose deliberately

- **Phase 1 (1928): Modernist / asymmetric / functional** — for modern, industrial, commercial, political, or poster work.
- **Phase 2 (post-1946): Classical / symmetric / humanist** — for books, literary publications, traditional or timeless work.
- **Rule:** Never mix phases within one artifact. Pick the register first, design second.
- If the brief is "modern poster for tech/industrial/avant-garde" → Phase 1.
- If the brief is "book, literary journal, classical brand, editorial long-form" → Phase 2.

---

## PHASE 1 — Die neue Typographie (1928)

### Asymmetric composition as default

- **Rule:** Centered layout is static, bourgeois, pre-modern. Asymmetric is dynamic, honest, functional.
- Weight distributes via tension, not mirroring. Never center a title unless the layout demands it for one structural reason.
- Optical balance via contrast: large left = small right, heavy top = light bottom.

### Sans-serif for functional work

- **Rule:** Use grotesque sans-serif for all modernist / functional / commercial work.
- Canonical faces: **Akzidenz-Grotesk, Futura, Univers, Helvetica, Neue Haas Grotesk, Inter.**
- Geometric sans (Futura, Avenir) for purely geometric work. Grotesque (Akzidenz, Helvetica) for general functional work.
- Avoid serifs entirely in Phase 1. Avoid decorative, ornamental, or humanist-script faces.

### White space as structural element

- **Rule:** White space is not background — it is a positive design element.
- Large uninterrupted white areas create visual rest and direct the eye.
- **Target: 40–60% of the modernist layout is whitespace.**
- Crowd elements deliberately to create contrast with the empty field.

### Geometric construction

- Elements align to a rational grid (see Müller-Brockmann). Never arbitrary placement.
- Horizontal and vertical lines dominate. Diagonals used sparingly for energy (political posters, dynamic events).
- Circles, squares, rectangles, bars — geometric shapes as structural support, not decoration.

### Function over ornament

- **Rule:** Every element must serve a communicative function. If it doesn't, delete it.
- No flourishes, no ornaments, no borders-for-the-sake-of-borders, no background patterns.
- Color: limited palette (black + 1 accent, usually **red**). Accents are functional, not decorative.

### Rational type choice and hierarchy

- Two weights of one sans-serif family (regular + bold) — sufficient for most work.
- Size hierarchy: use large size jumps (2–4× between levels) for clear information rank.
- Avoid italic in Phase 1 — italics are humanist-script relics. Use weight + size instead.
- **ALL CAPS sparingly** — only for short labels, never for body or headlines.

### Typography dynamics

- Flush-left, ragged right — always. Justified text is a humanist relic in Phase 1.
- Ultra-tight or ultra-loose tracking for display type, based on intent.
- Baseline alignment across columns enforced.

### Phase 1 anti-patterns — reject

- **Reject:** centered layouts (except for single-element logos where centered is the only structural option).
- **Reject:** serif typefaces for body or display.
- **Reject:** italic, decorative, blackletter, script faces.
- **Reject:** ornament, borders, rules of decorative character, curly flourishes.
- **Reject:** dense full-page layouts (loss of white space = loss of structure).
- **Reject:** earthy/pastel palettes — use pure primaries, or black + red accent.

## PHASE 2 — The Form of the Book (1975)

### Symmetric page architecture

- **Rule:** For books and classical work, return to centered symmetry on the page. Text block centered on the spread.
- Two-page spread is the unit of composition, not the single page.
- Mirror margins across the gutter (inner margin = inner margin on facing page).

### Classical serif typefaces

- **Rule:** Use classical serifs with historical lineage.
- Canonical faces: **Garamond, Caslon, Baskerville, Bembo, Jenson, Centaur, Sabon** (Tschichold's own, 1964).
- Avoid geometric sans and grotesque in Phase 2 work — wrong register entirely.
- One serif family for body + display. Italic for titles, foreign words, emphasis.

### Golden-section proportions

- **Page proportion**: 1 : 1.618 (golden section) OR 2:3 (medieval canon) OR 5:8 (Fibonacci).
- **Text block proportion**: same as page proportion (golden rectangle inside golden page).
- **Rule:** The text block is ALWAYS smaller than the page — generous margins are essential.

### Margin ratios — the classical canon

- **Tschichold's classical ratio: inner : top : outer : bottom = 2 : 3 : 4 : 6.**
- Inner margin smallest (facing-page gutter absorbs some space).
- Bottom margin largest (visual foot anchors the text block).
- Outer margin > top > inner. Always.
- For octavo / A5 / 6×9" book: typical margins = 15mm / 22mm / 30mm / 45mm.

### The Van de Graaf canon

- Medieval page-construction method predating Tschichold, rediscovered by him.
- Page divided into 9 parts horizontally AND vertically.
- Text block positioned so:
  - Text block height = page width
  - Inner margin = 1/9 page width
  - Top margin = 1/9 page height
  - Outer = 2/9, Bottom = 2/9
- Produces the 2:3:4:6 ratio geometrically.
- **Rule:** For genuine book design, apply Van de Graaf construction. Never "eyeball" book margins.

### Body type settings

- Size: **10–12pt** body text for books.
- Leading: **12–15pt** (120–125% of body size).
- Measure: **60–70 characters** per line.
- Justified with hyphenation enabled. Ragged right is NOT classical.
- Small caps + old-style figures (text figures) for drop-caps, chapter openings.

### Classical ornament — permitted, barely

- **Rule:** Ornament permitted only if historical: fleurons (printer's flowers), rule lines, small caps flourishes.
- Used at chapter breaks, title pages — sparingly.
- No modernist geometric shapes in Phase 2 work.

### Phase 2 anti-patterns — reject

- **Reject:** asymmetric layouts (wrong register).
- **Reject:** sans-serif body type (acceptable for captions or imprint only).
- **Reject:** red accent colors, bold primary palettes.
- **Reject:** tight margins, dense packing, edge-to-edge text.
- **Reject:** ragged right for body text.
- **Reject:** computer-default margins (Word's 1-inch default is never correct).

## Choosing between phases — decision table

| Brief keyword | Phase |
|---|---|
| Poster, flyer, political, tech, industrial | 1 (modernist) |
| Corporate identity, functional brand | 1 |
| Book (novel, essay, poetry, academic) | 2 (classical) |
| Literary journal, catalogue raisonné | 2 |
| Modernist art exhibition poster | 1 |
| Classical music program, opera booklet | 2 |
| Product label, consumer packaging | 1 (usually) |
| Wedding invitation, formal announcement | 2 |

## Quick validation checklist

1. Is the phase (1 or 2) declared before design begins?
2. Phase 1: sans-serif throughout? Asymmetric? Red + black only? White space ≥ 40%?
3. Phase 2: classical serif? Symmetric spread? Van de Graaf or 2:3:4:6 margins? Justified body 60–70 chars?
4. Phase 1: any italics, ornaments, centered titles? Remove.
5. Phase 2: any sans body, ragged right, tight margins? Fix.
6. Is the text block proportion matching the page proportion (golden rectangle inside golden page)?
7. Phase 2 books: bottom margin largest? Inner smallest?
8. Could the artifact be mistaken for the wrong phase? If yes → sharpen the register.
