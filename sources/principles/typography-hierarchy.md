# Typography Hierarchy — Working Principles

## Hierarchy = size + weight + colour + space
Для читаемой иерархии нужны MINIMUM 3 уровня:
1. **Primary** (headline / hero)
2. **Secondary** (subtitle / section heading)
3. **Body** (supporting text / details)

Между levels должен быть **clear contrast** (не "чуть больше" — а **visibly different**).

## Size ratios (modular scale)
Используй модульный scale, не произвольные размеры:
- Minor third (1.2) — subtle, editorial
- Major third (1.25) — balanced, common
- **Perfect fourth (1.333)** — punchy, recommended для posters
- **Golden ratio (1.618)** — dramatic, editorial
- Examples базой 16px: 16 → 21 → 28 → 37 → 49 → 65 → 86

## Font pairing — ONE serif + ONE sans, или ONE display + ONE neutral
- НЕ 3+ разных шрифтов (unless you're Josan Gonzalez doing maximalist)
- Classic pairings:
  - **Impact / Helvetica** (display + neutral)
  - **Playfair / Source Sans** (serif + sans)
  - **Eurostile / Courier New** (geometric + mono, cyberpunk-friendly)
  - **Druk / GT America** (display condensed + clean sans)

## Letter-spacing (tracking)
- **Headlines в uppercase** — positive tracking (50-200 units)
- **Body text** — neutral or slight negative (−10 to 0)
- **Headlines в lowercase** — tight (−20 to −50 units)
- **Никогда не делай** heavy tracking на body — снижает legibility

## Line-height (leading)
- **Body text:** 1.4 — 1.6
- **Headlines:** 1.0 — 1.2 (tight)
- **Display type:** 0.9 — 1.0 (can overlap)

## Measure (line length)
- Body: **45-75 characters per line** — оптимально
- Headlines: as wide as needed, не ограничивай

## Alignment choices
- **Left-align** — default, natural, most legible
- **Center** — for short headlines, titles, posters — НЕ для body
- **Right-align** — rarely, for accent / callout
- **Justified** — ТОЛЬКО если правильно hyphenated, иначе ugly rivers

## Weight contrast
- Heavy weights (700-900) — for PRIMARY impact only
- Mix: **Heavy hero + light subtitle** — strong hierarchy
- Avoid всё bold — bold devalues when overused

## Display type как графический элемент
- В posters типографика часто IS главный визуальный элемент
- Use: kerning adjustments, negative/positive space balance, letter as shape
- RGB split / offset / glitch — современная display тенденция

## Type on path (assignment requirement)
- Короткие фразы по curve — good
- Длинные sentences по path — obnoxious
- Path should make sense (arc, wave) — не arbitrary

## Type mask (clip-path / text-fill)
- Text shape как маска для image/pattern — **impactful**
- Works best для bold weight, large size, condensed letterforms
- Keep mask text short (1-3 words, numbers) — или unreadable

## Cyberpunk typography patterns
- **Ultra-condensed bold** (Impact-like) для hero
- **Courier / mono** для tech readouts / data
- **Kanji / katakana** как decoration (cultural signifier)
- **RGB split** — pink + cyan offset behind white
- **Wide tracking** в UPPERCASE headlines
- **Scanlines** crossing through type для CRT-monitor feel

## Font resources
- **Google Fonts** — free, большой выбор
- **Adobe Fonts** — premium, бесплатно с CC
- **Fontshare** — free quality fonts (Indian Type Foundry)
- **Befonts / Pixel Surplus** — freebies

## Common mistakes
1. Слишком много фонт-семейств в одной работе (>2)
2. Недостаточный contrast между levels (hierarchy fails)
3. Body text размер <14pt на печатном postеr — unreadable
4. Bold уже везде = nothing bold
5. Justified body без hyphenation control
6. Centered body text (беззубый, mimics wedding invitations)
