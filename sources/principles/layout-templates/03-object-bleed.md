# Template 03 — Object Bleed

*Single iconic object dominates, bleeding off one edge of the frame. Type wraps around or sits in the dead space. Record covers, product features, symbolic artifact posters.*

## When to use

- **Album covers** featuring the physical artifact (vinyl, cassette, CD)
- **Product launch** posters where the object IS the news
- **Symbolic artifact** posters (a key, a mask, a hand, a weapon)
- **Museum object** posters (single artifact feature)
- **Book covers** when the object is more evocative than the author
- **Wine / spirits / cosmetics** — hero bottle or vessel bleeding

Do NOT use when:
- Multiple objects to feature (use Template 01 with scene composition)
- Person/figure is the subject (use Template 01 portrait)
- Abstract non-objectal subject (use Templates 02 or 04)

## Grid plan

```
viewBox: 612 × 888 OR 612 × 612 (square album)
Outer frame: x=28→584, y=28→860 (thin rule)
Safe content zone: x=56→556
Object bleed axis: RIGHT (most common) OR BOTTOM (vertical emphasis)
```

## Vertical zones (bleed-right variant)

| y-anchor | Zone | Content |
|---|---|---|
| y=56 | Top meta line | Publisher / label left, catalog right |
| y=90–150 | Wordmark | Title treatment (classical serif, small-medium) |
| y=120 | Rule under wordmark | Thin accent line |
| y=144 | Tagline | Italic serif, oxblood or accent color |
| y=150–550 | **OBJECT HERO** (cx > 306, extends past x=556) | Large object with right edge clipped by frame |
| y=200–350 | Text column (left of object) | Artist / author / metadata column |
| y=380–500 | Secondary text | Details / ensemble / credit block |
| y=620 | Rule divider | — |
| y=640–720 | Feature list | Tracks / chapters / contents in 1-2 columns |
| y=750 | Recording meta | Location / date / technical details |
| y=800–860 | Footer | Standard footer credits |

## Object placement math

For bleed-right, object:
- Center x > 306 (shifted right)
- Right edge > 556 (bleeds past safe zone) AND > 584 (bleeds past frame)
- Example: `cx=420, r=230` → right edge = 650 (bleeds past 584 frame = off-page)
- Left edge should still be visible: 420 - 230 = 190 (comfortable in frame)

Don't center the object — the asymmetry is the point.

## Required content blocks

1. **The object** (bleeding off right OR bottom)
2. **Wordmark / title** (in the non-bleed quadrant, small-medium)
3. **Text column** in the clear quadrant (left or top) with metadata
4. **Footer** with credits

## Optional content blocks

- Hand-drawn scribble / mark (Sagmeister craft accent)
- Accent rule in saturated color
- Feature list (tracklist / chapter list / ingredient list)
- Recording / production meta line
- Edition stamp

## Compatible style profiles

- **Peter Saville** — iconic artifact variant (Blue Note revival, Factory)
- **Stefan Sagmeister** — craft-visible variant with hand-drawn marks
- **Kenya Hara** — single vessel on warm-white ground variant
- **Beeple** — digital artifact variant (3D object feel)
- **Massimo Vignelli** — Swiss-disciplined object presentation
- **Kilian Eng** — a ship / vehicle / creature bleeding off

## Compatible trend layers

- **Editorial Maximalism** — bleed IS the maximalism move
- **Dark Academia** — classical object with serif typography
- **Neo-Futurism** — frosted glass over object
- **Dreamcore / Liminal** — object in muted dreamy atmosphere

## Object construction in SVG

For a vinyl record (most common Object-Bleed example):
- Black disc with ~20 concentric groove rings at varying stroke-widths
- Center label (red with gold ring border)
- Spindle hole (small circle, paper color)
- Sheen overlay (diagonal white gradient at low opacity)
- Soft shadow layer beneath

For a bottle / vessel / mask:
- Silhouette from paths with gradient fill for 3D feel
- Subtle specular highlight
- Reflected light on opposite side
- Ground shadow ellipse below

For a classical hand / artifact (Saville appropriation):
- Painterly gradient for depth
- Low-contrast edges bleeding into paper
- One warm highlight (amber/ivory)
- Canvas/grain texture overlay

## Anti-patterns for this template

- Object perfectly centered with even margins (kills the bleed tension)
- Object completely within frame (not a bleed anymore)
- Multiple objects competing (pick ONE)
- Object too small relative to canvas (the point is dominance)
