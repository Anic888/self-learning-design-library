# Template 02 — Type-as-Image

*Typography IS the visual. No imagery or imagery reduced to decoration. The headline word or numeral dominates the canvas. Peter Saville, Paula Scher, Vignelli, Kompakt lineage.*

## When to use

- **Anniversary compilations** (the year IS the message)
- **Music events** where the name is the brand
- **Editorial covers** — magazine headlines
- **Concept posters** — single-word ideas
- **Label identity** work (Factory Records, Kompakt, Blue Note covers)
- **Campaign posters** with strong verbal punchline
- **Artist-less commercial** (not a person, not an image — just a name)

Do NOT use when:
- Subject is a specific person (use Template 01 with their portrait)
- Abstract atmospheric subject (use Template 04 Field)
- Product needs to be shown physically (use Template 03 Object-Bleed)

## Grid plan

```
viewBox: 612 × 888 (vertical) OR 888 × 612 (horizontal) OR 612 × 612 (square album)
Outer frame: x=28→584, y=28→860 (optional thin rule)
Safe content zone: x=56→556
```

## Vertical zones

| y-anchor | Zone | Content |
|---|---|---|
| y=56 | Top meta | Label/studio left, catalog number right |
| y=68 | Hairline (optional) | — |
| y=110–160 | Wordmark / small title | Secondary verbal element (label name, over-title) |
| y=180–450 | **HERO TYPE** (270 tall OR larger) | Giant numeral / word / cropped word |
| y=460–520 | Tagline + date block | Year range, subtitle, tagline |
| y=548–780 | Feature grid | 6-12 names (speakers, artists, tracks) in 3-col |
| y=800 | Footer hairline | — |
| y=820–856 | Footer credits | Publisher / pressing / catalog |

## Hero type variations

### Variant A — Giant Numeral (Kompakt 25 style)
- Single huge number or 2-digit year, 340pt font-size, text-anchor middle at x=306
- Accent line through it (Factory yellow horizontal rule)
- Occupies 40% of vertical canvas

### Variant B — Cropped Word Bleeding (Scher Editorial Maximalism)
- Word in 200-280pt, positioned with x<0 or x>viewport so it bleeds off edge
- Remaining word shapes become visual elements
- One saturated accent color (hot orange, electric blue)

### Variant C — Stacked Weight-Variable Wordmark (Saville Factory DNA)
- 2-3 words stacked, extreme weight contrast (bold + italic, condensed + extended)
- All caps with wide tracking OR mixed case with tight tracking
- Minimal color — monochrome + one accent

### Variant D — Single Letter as Monument
- One giant letter 400–550pt
- Sub-type describing what the letter means
- Works for abstract identity posters

## Required content blocks

1. **Hero type** (takes 40-70% of canvas)
2. **Top or bottom meta line** with catalog/ID/edition
3. **Footer** with publisher/studio/date

## Optional content blocks

- Small wordmark above or below hero type
- Saturated accent (Factory yellow rule, red dot, iridescent gradient)
- Numbered list of features (speakers, tracks, artists) in grid below hero
- Serial number stamp (Factory convention)
- Small classical divider ornaments

## Compatible style profiles

- **Peter Saville** — native fit (Factory Records DNA)
- **Paula Scher** — cropped-word variant (Public Theater lineage)
- **Massimo Vignelli** — monumental numeral variant (Swiss discipline)

## Anti-patterns for this template

- Multiple competing giant words (pick ONE hero)
- Hero type with photograph alongside at similar size (that's Template 01)
- Tiny hero type (that's Template 04 or back to 01)
- Fully centered perfectly-balanced composition (it gets dull — always have asymmetry or scale-extreme)
- All caps with uniform size (needs weight / size contrast)

## Typography musts

- At least 2 distinct weights or widths (weight variation = life)
- ONE saturated accent color max
- Bringhurst-compliant real glyphs (em-dash, curly quotes)
- Classical or grotesk — don't mix too many eras of type classification
- Generous negative space AROUND the hero type
