# Color Theory — Working Principles

> Не академия, а applied rules для design composition.

## Limit the palette
- **3-4 colors max** для most work. 5 — потолок. Больше — сырой дизайн.
- Assignment requirement commonly asks for "limited palette" — это универсальное good design правило, не ограничение.

## Role-based distribution (60-30-10)
- **60%** — dominant (обычно background / large areas)
- **30%** — secondary (main content elements)
- **10%** — accent (CTA, highlights, focal attention)
- 60-30-10 — не догма, но ПРОПОРЦИЯ важнее конкретных значений. Никогда не делай 4 цвета по 25%.

## Contrast ratios (accessibility + reading)
- Body text vs background: ≥ **4.5:1** (WCAG AA)
- Large headlines: ≥ **3:1**
- Decorative only: can be lower but beware illegibility
- **Инструмент:** WebAIM contrast checker

## Color temperature as emotional language
- **Warm (reds/oranges/yellows)** — energy, urgency, human
- **Cool (blues/cyans/greens)** — calm, technology, distance
- **Neutrals (greys/browns)** — balance, industrial
- **Contrasting temps** — tension / focal point

## Saturation as hierarchy
- Saturated colors → **read first** visually
- Use saturation sparingly для main focal elements
- Backgrounds / support → lower saturation
- Common mistake: everything equally saturated = nothing important

## Monochromatic discipline
- Monochromatic (one hue + variations in saturation/lightness) — **often looks more refined** than multi-hue
- Add ONE accent color if needed (complementary или triadic relationship)
- Rule: Thorp-style = monochromatic + one warm accent

## Gradients
- Linear gradients: 2-3 stops max. Больше — looks amateur.
- Radial gradients: для light sources / focal glow
- **НЕ random gradients** — gradient должен иметь meaning (light source, depth, movement)

## Cyberpunk-specific palette patterns
- **Neon noir:** near-black + hot pink + electric cyan (3 colors)
- **Synthwave:** magenta→purple→blue gradient + orange sun + cyan grid
- **Dense cyberpunk (Josan):** pink + cyan + yellow + black + white (5 colors)
- **Ghost in the Shell (Thorp):** black + white + one amber accent

## Color tools
- **Coolors.co** — palette generation
- **Adobe Color** — color wheel exploration
- **Khroma** — AI-personalized palettes
- **Paletton.com** — classic color theory wheel

## Common mistakes
1. Too many saturated colors competing for attention
2. Using pure black (#000) instead of dark blue/brown (looks harsh, cheap)
3. Pure white (#FFF) on pure black (#000) — too clinical, prefer warm off-whites on deep navies
4. Gradient without direction/meaning
5. Ignoring contrast for body text
