# Composition & Grids — Working Principles

## Start with a grid
- **Don't design без grid.** Даже если потом breakable — начни с structure.
- **12-column grid** — universal default (web, print, posters)
- **Columns with gutters** — е.g., 12 columns @ 60px + 20px gutter

## Focal point discipline
- **ONE primary focal point** per composition. Не два равных competing elements.
- Glance test: close eyes, open → что первое привлекает взгляд? That's your focal.
- Support elements должны направлять взгляд К focal, не конкурировать.

## Rule of thirds
- Divide canvas 3×3. Place focal на intersection.
- **Не на center** (unless asymmetric would break mood) — center-compositions часто static и boring.
- Для poster 8.5×11: горизонтальные trisections на y ≈ 264, 528. Вертикальные на x ≈ 204, 408.

## Asymmetric vs symmetric
- **Asymmetric** — modern, dynamic, harder (требует balance через visual weight)
- **Symmetric** — classic, formal, safer (wedding invites, official notices)
- **Radial** — for specific effects (mandala, spotlight)
- Default для poster: **asymmetric balanced** — один heavy side offset лёгким другим

## Visual weight
Что делает элемент "heavy":
- Size (large = heavy)
- Colour saturation (saturated = heavy)
- Colour contrast vs background (high contrast = heavy)
- Density (detailed = heavy)
- Position (lower = heavier, bottom gravity)
- Isolation (alone = heavy due to attention)

Balance heavy element on one side с multiple lighter elements на другой.

## Negative space как активный element
- Не бойся пустоты. Пустота фокусирует.
- **30-50% negative space** — good default для modern poster
- Minimalism = NOT = lazy. Every empty area должно быть intentional.

## Breathing room / margins
- Outer margins: **minimum 20-40px** even на small posters
- Never let content touch edge (unless bleed-intentional)
- Internal padding: **consistent multiples of 8** (8, 16, 24, 32, 48, 64)

## Visual flow / eye path
Люди читают слева-направо, сверху-вниз (Western):
1. **Top-left** — где начинается взгляд
2. **Center** — где задерживается
3. **Bottom-right** — где завершается

Compose сompelling path:
- Top-left: brand / stamp / date
- Center: hero / main message
- Bottom-right: CTA / venue / signature

## Golden ratio (φ = 1.618)
- Ratio 1 : 1.618 часто выглядит "natural"
- Для poster: split vertically 61.8% / 38.2% — отличная proportion
- НЕ догма — но хороший fallback когда не уверен в proportions

## Gestalt principles (быстрый cheat sheet)
- **Proximity** — близкие элементы воспринимаются как группа
- **Similarity** — похожие (цвет/форма/размер) — группа
- **Closure** — неполные формы mind completes
- **Continuity** — взгляд следует плавным линиям
- **Figure-ground** — differentiate between object & background

## Layers of depth
Добавь **3-5 visual planes**:
1. Background (atmospheric)
2. Mid-ground (contextual)
3. Foreground (focal)
4. Overlay (texture, scanlines, vignette)
5. Top UI (timestamps, labels, marks)

Depth creates **interest without clutter**.

## Repetition and rhythm
- Repeat один visual element 3+ раз — creates pattern
- Vary size / position / opacity в repetition для rhythm
- Isolated single element = accent. Repeated = motif.

## Common poster composition structures
1. **Centered hero** — type + imagery on axis (Signalnoise default)
2. **Asymmetric focal** — hero offset, support on opposite side (editorial)
3. **Stacked bands** — horizontal zones top/middle/bottom (magazine covers)
4. **Grid-driven modular** — content boxes in grid (Swiss design)
5. **Immersive illustration** — edge-to-edge art, type overlaid (movie posters)

## Cyberpunk-specific composition patterns
- **Central figure + packed environment** (Josan)
- **Horizon-anchored wide vista + HUD corners** (Thorp)
- **Sun + grid + centered type** (Signalnoise)
- **Big one-word type + minimal support** (brutalist cyberpunk)

## Common mistakes
1. Symmetric composition used by default (boring)
2. Too many focal points competing
3. No negative space (cluttered)
4. Content touching canvas edge
5. Random positioning without grid underneath
6. Heavy elements stacked on one side (unbalanced)
7. All elements same size (no hierarchy)
