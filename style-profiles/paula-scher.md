# Paula Scher — Style Profile

## Core Identity
American typographic maximalism. **Type as image, cropped aggressively, colored with conviction**. The Public Theater identity is her manifesto — wood-type letterforms so oversized they become patterns. Political-feeling even when not political. Opposite of Swiss restraint, and opposite of Carson's chaos too: Scher is deliberate, confident, bold.

## Palette
- `#F05A28` / `#E8540F` — signature hot orange (Public Theater accent, 15-30%)
- `#0E0E0E` — near-black (dominant typography, 25-40%)
- `#F5F5F0` — off-white / cream (ground, 20-30%)
- `#D61A3C` / `#C11A3C` — deep red (when contrast needed, 10-20%)
- `#176AA3` / `#1A5084` — strong saturated blue (occasional, 5-15%)
- `#F5E04A` / `#E8D800` — acid yellow (pop accent)

**Rule:** **2-4 SATURATED colors on each piece**, no subtle washes. Colors fight/complement like gang patches. Unlike Thorp's discipline (mono+1) or Signalnoise's harmonic gradients, Scher uses colors as independent SHOUTS.

## Typography
- **WOOD-TYPE CHUNKY SLAB SERIFS** — signature move (Rockwell, Clarendon, custom wood-block forms)
- Heavy sans display (Knockout, Futura Bold, Helvetica Black) — alternative
- **Custom hand-lettering** frequently, painted/brushed feel
- ALL CAPS dominant for hero type
- **Tight kerning / letters touching** — often overlapping
- **Typography at massive scale** — letters crop off the edge, become patterns
- Secondary text: small, structural, supports hero type
- **Mixed weights** of same family in one piece (bold + light together for contrast)

## Composition & Grid
- **Grid-aware but overrides it** — modular system exists but type bleeds off edges
- **Type IS the composition** — letterforms become architectural elements
- **Asymmetric, weighted, off-balance** — intentional compositional tension
- **Cropped letterforms** — one huge letter occupies 60% of space, partial
- **Color-block layering** — flat shapes as backgrounds for type
- Central axis avoided — things push to corners, edges

## Signature Moves
- **Oversized cropped letters** — one letter occupies majority of composition
- **Wood-type slab serif** — Public Theater DNA
- **Flat color blocks as background** for type
- **Type stacks** — multiple lines at different sizes stacked tight
- **Painted hand-feel** — letters look like they were brushed not set
- **Typographic maps** — continents/cities filled with place names (her personal paintings)
- **High-saturation 3-4 color palette** per piece
- **Political/cultural weight** — even commercial work feels like it has something to say
- **Physical materiality** — letterforms feel like they could be wooden blocks, not vectors

## Techniques / Tools
- Illustrator (vector type manipulation)
- Hand-lettering / brush painting (later scanned)
- Historically: actual wood type + letterpress (Public Theater inspiration)
- Photoshop for texture/finish
- Large-scale printing expertise (subway station murals, building graphics)

## Mood / Emotional Register
Confident, declarative, energetic. "This is important and I'm saying so." Political-adjacent even in commercial work. Never apologetic, never subtle. American pop-commercial maximalism.

## When to apply
- ✅ Cultural institution identity (theater, museum, ballet)
- ✅ Political / cause / activism campaigns
- ✅ Book covers needing strong type presence
- ✅ Wayfinding / environmental graphics
- ✅ When project needs "loud but literate"
- ❌ NOT for corporate conservative contexts
- ❌ NOT for minimal / understated
- ❌ NOT for image-heavy illustration projects (she's typography-forward)

## How to apply (tactical for Claude)
1. **Typography first** — design the title/hero type before anything else. It IS the composition.
2. **Crop letters aggressively** — let one letter bleed off edge, occupy 50-70% of canvas.
3. **Wood-type slab serif** as default headline family — Rockwell, Clarendon, Cooper Black. NOT clean Helvetica.
4. **Palette: 2-4 saturated independent colors**. Hot orange + black + off-white is her greatest-hits baseline.
5. **Flat color blocks** behind type — rectangular shapes, not gradients.
6. **Multiple text sizes stacked** — hero huge, caption tiny, ratio 10:1 or more.
7. **Hand-feel** if hand-lettering — slight imperfection, brush texture.
8. **Bleed intentionally** — things run off edges.
9. **One statement per piece** — don't split focus across multiple hero elements.
10. **NOT to use:** centered balanced composition, gradient backgrounds, image-heavy layouts, subtle palettes, delicate/thin typography. All these fight Scher's voice.
