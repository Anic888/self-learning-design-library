# James White / Signalnoise — Style Profile

## Core Identity
80s retrofuturism through graphic design lens. Synthwave aesthetic with broader reach than "just" synthwave — substantial character illustration, brand/game work (Blood Dragon, Twitch, Cartoon Network), posters. Unifying signature: **deep violet base + prismatic neon accents + geometric framing**.

## Palette
**Verified live via signalnoise.com 2026-04:**

- `#1E0B3A` / `#2A1550` — **deep violet/indigo** (base — not pure black, this matters)
- `#F72585` / `#FF1E8A` — hot magenta/pink (primary accent, 15-25%)
- `#FF6D00` / `#FF8C1F` — vibrant orange (secondary accent, 10-20%)
- `#00D9FF` / `#2EFFF0` — electric cyan/mint (light accents, 5-15%)
- `#FEE440` / `#FFD500` — acid yellow (pop highlights, 5%)
- `#7209B7` / `#9A4DFF` — bright purple (mid tones, 5-10%)

**Rule (refined from live observation):**
- Background: deep violet/indigo — **almost never pure `#000`**
- Multiple accent hues (4-5 common) used in structured relationship, often as gradient stops
- Gradients are mandatory — synthwave-standard magenta → purple → cyan transitions
- Chrome highlights (near-white) used sparingly for hero type

## Typography
- **Heavy display serif/sans** (Druk, Eurostile Extended, custom geometric)
- Custom chrome/neon lettering on hero words — metallic gradient fills, occasional italic slant
- **80s "ski goggle" italic** — dramatic slant on key words
- **Drop shadow + outline** stacked treatments
- Secondary typography often **all-caps technical sans** for taglines
- Hand-painted / custom script on music/entertainment work
- Serif slabs frequent for title treatment (opposite of Thorp's condensed light)

## Composition & Grid
- **Centered/symmetric** more than asymmetric (poster convention)
- **Horizon-anchored** — low horizon, huge sky with sun/moon
- **Perspective grid floor** on landscape pieces (receding pink/cyan lines)
- **Geometric framing** — triangles, circles, hexagons, chevrons as container shapes (verified: triangle frames around desert dune pieces)
- **Hero character/object** dead-center in character illustration work

## Signature Moves
- **Sunset gradient sky** (magenta → purple → indigo) — almost always
- **Perspective grid floor** — pink/cyan lines receding to single vanishing point
- **Sun/moon circle** in upper third, often gradient-filled
- **Chrome typography** — metallic gradients with highlight reflections
- **Geometric badge framing** — hex/triangle/circle containers around main elements
- **Mountain/pyramid silhouettes** in middle distance
- **Palm trees** (Miami 80s) when tonally appropriate
- **Character illustration** with neon rim-lighting and prismatic accents
- **Chrome characters** — mirrored/metallic hero figures (verified via live portfolio)
- **Dynamic diagonal compositional energy** on action pieces
- **Gradient on every surface** — rarely flat fills

## Techniques / Tools
- Illustrator (primary — vector poster work)
- Photoshop (finishing: grain, glow, chromatic aberration)
- Custom hand-lettering / chrome type generation
- Strong texture overlay work (film grain, CRT scanline when appropriate)

## Mood / Emotional Register
Nostalgic optimism. "The future we were promised in 1985." NOT dystopian, NOT ironic — sincerely romantic and celebratory. Always energetic, never melancholic. Joyful maximalism (opposite of Thorp's meditative minimalism).

## When to apply
- ✅ Synthwave / vaporwave / retrowave context
- ✅ Indie electronic / chiptune music
- ✅ 80s-influenced game / entertainment branding
- ✅ Summer events, nostalgia-themed work
- ✅ Character hero art with retro sci-fi tone
- ✅ When energy + romance needed, not gravitas
- ❌ NOT dark cyberpunk (different universe — that's Thorp/Josan)
- ❌ NOT corporate editorial (too emotive)
- ❌ NOT minimalist (Signalnoise is maximalist)

## How to apply (tactical for Claude)
1. **Base color: deep violet (#1E0B3A or #2A1550), NOT black.** This is a critical tell.
2. **ALWAYS use gradients.** Sunset palette (magenta → purple → indigo, or orange → pink → purple) as default sky/background.
3. **Horizon at ~60-70% from top.** Large sky area for atmospheric color gradient.
4. **Perspective grid floor** — pink or cyan lines receding to central vanishing point, especially for landscape compositions.
5. **Sun/moon circle** in upper half, gradient-filled.
6. **Geometric framing device** — triangle/hexagon/circle container around hero element.
7. **Chrome-style typography** on main title — metallic gradient fills, optional italic slant.
8. **Hero element gets rim-lighting in neon** (magenta + cyan bounces).
9. **Mountain/pyramid silhouettes** in middle distance for depth.
10. **Grain overlay** (film noise) for 80s tactility.
11. **4-5 colors in relationship** — never just 2. Include magenta + cyan + orange + yellow + violet combinations.
12. **NOT to use:** hard cyberpunk dystopian messaging, pure black backgrounds, thin-line minimalism (Thorp zone), flat non-gradient fills, Courier/mono typography as dominant.
