# James White / Signalnoise — Style Profile

## Core Identity
80s retrofuturism through graphic design lens. Synthwave aesthetic — но с sensibility graphic designer, не illustrator. Geometric, clean, poster-forward.

## Palette
- `#F72585` → `#7209B7` → `#3A0CA3` — sunset gradient (magenta → purple → deep blue, 30-50%)
- `#FF6D00` / `#F48C06` — orange (rising sun, 10-20%)
- `#00B4D8` / `#00F5FF` — cyan (horizon, grid, 5-10%)
- `#FFFFFF` — chrome highlights (5%)
- **Тенденция:** ОБЯЗАТЕЛЬНЫЙ gradient (почти никогда flat), trivial trio pink/purple/blue или orange/pink/purple.

## Typography
- **Headlines:** Heavy display serif (like Druk, Geo, Eurostile Extended) ИЛИ script signature
- **Chrome / neon lettering** — custom-drawn часто
- **80s italic slant** — "ski goggle" типографика
- **Drop shadow + outline** layered treatments
- **Body:** Minimal — postcards/prints обычно без больших блоков текста

## Composition & Grid
- **Symmetric / centered** чаще чем asymmetric
- **Horizon-anchored** — низкая линия горизонта, массивное небо
- **Perspective grid** снизу (не всегда но часто)
- **Geometric shapes** — triangles, circles, chevrons как фреймирующие элементы

## Signature Moves
- **Sunset gradient sky** (magenta → purple → navy) — almost always
- **Perspective grid floor** — receding pink/cyan lines
- **Sun/moon circle** в верхней трети
- **Chrome typography** — metallic gradients на type
- **Geometric "badge" framing** — hexagons, triangles, circles как containers
- **Mountain silhouettes** — простые geometric mountain shapes
- **Palm trees** для topical 80s Miami vibe (когда уместно)

## Techniques / Tools
- Illustrator (primary — vector poster work)
- Photoshop для finishing (grain, glow)
- Hand-lettering для custom type

## Mood / Emotional Register
Nostalgic optimism. "Future we were promised." Не dystopian, не ironic — искренне romantic.

## When to apply
- ✅ Synthwave / vaporwave контекст
- ✅ Music: indie electronic, retrowave artists
- ✅ Summer events, 80s nostalgia
- ✅ Когда нужна "Instagram-friendly" энергия без сарказма
- ❌ НЕ подходит для dark/serious cyberpunk (Blade Runner) — это другая вселенная
- ❌ НЕ для corporate/editorial (слишком эмотивно)

## How to apply (tactical для Claude)
1. **Палитра: ВСЕГДА gradient.** Sunset palette (magenta → purple → deep blue) как default.
2. **Horizon на ~60% высоты сверху** — много неба.
3. **Perspective grid floor** — pink или cyan линии, convering в центральную точку на горизонте.
4. **Sun/moon** — большой круг в верхней половине. Часто с gradient fill.
5. **Геометрические обрамления** — hexagonal или triangular badge вокруг main type.
6. **Chrome-style типографика** — metallic gradient fill на headlines.
7. **Добавь grain / film noise overlay** для 80s тактильности.
8. **НЕ использовать:** hard cyberpunk messages, dystopian imagery, dense illustration (это Josan). Signalnoise искрений и clean.
