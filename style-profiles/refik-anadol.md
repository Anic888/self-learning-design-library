# Refik Anadol — Style Profile

## Core Identity
**Machine-imagination aesthetics rendered at architectural scale**. AI-generated flowing forms — latent space made visible. Point clouds, neural hallucinations, data dreams. Anadol's work looks like what StyleGAN dreams when directed by someone with taste. Reference for designers working in AI-adjacent visual territory.

## Palette
Variable — palette often derived from source dataset (cultural imagery, weather data, brain waves). Tendencies:

### Mode 1 — Neon Data
- `#00F0FF` / `#00D9FF` — electric cyan (primary)
- `#FF0080` / `#F72585` — magenta
- `#FFEE00` — bright yellow
- `#0E0428` / `#1A0A3A` — deep purple-black base
- Saturation: **extreme**. Colors are glowing, not painted.

### Mode 2 — Architectural Monochrome
- Muted metallic grays
- One accent (often amber or cyan)
- Concrete/glass palette — reflections of architectural context

### Mode 3 — Organic Data
- Green/teal biological tones
- Warm earth (when data source is natural — winds, flora, fauna)
- Soft gradients of related hues

**Rule:** palette is data-driven. Source dataset dictates color (MoMA paintings → MoMA painting colors, winds data → atmospheric tones).

## Typography
- **Minimal — often absent**. Work is experienced as video/installation, type is metadata
- When present:
  - **Technical sans** (Neue Haas Grotesk, Akkurat, or custom mono)
  - Small sizes, functional
  - Gallery-caption typography for print documentation
- NOT a typography-forward practice — focus is on generated imagery

## Composition & Grid
- **Architectural scale** is defining — work exists as huge projected surface, then documented in video
- **Centered or symmetrical** when projected on buildings
- **Irregular / flowing** when on free surface (Serpentine pavilion)
- **Time-based** — composition evolves over minutes (not static)
- Still documentation: **cinematic 16:9 or vertical installation proportions**
- Point cloud formations typically fill frame edge-to-edge

## Signature Moves
- **Flowing point cloud forms** — clouds of data points in motion
- **StyleGAN-era latent interpolation aesthetic** — smooth morphing between states
- **Dataset-as-subject** — MoMA collection, Istanbul weather, human brain waves as raw material
- **Architectural integration** — projected onto buildings, merged with venue
- **Saturated glow** (Mode 1) — neon-bright color emitting light
- **Slow temporal evolution** — video pieces unfold over 10+ minutes
- **No recognizable imagery** — abstract patterns, not figures/landscapes
- **Reflection / refraction** — imagery often moves through imagined liquid/atmospheric space
- **Sound synchronization** — visuals pulse/breathe with audio

## Techniques / Tools
- **Custom ML models** (often StyleGAN / StyleGAN2 / StyleGAN3 trained on curated datasets)
- NVIDIA GPUs for inference at resolution
- **Notch / TouchDesigner / custom Unreal** for real-time projection rendering
- Audio-reactive programming for installation sync
- Large-scale projection mapping (Christie, Barco)
- Team includes ML researchers, 3D artists, sound designers

## Mood / Emotional Register
Sublime-technological, contemplative, spectacular. "Look what the machine dreams." Not cynical like Beeple, not melancholic like Thorp — **awe-inducing**. Gallery-scale wonder.

## When to apply
- ✅ AI-adjacent art / tech-forward contexts
- ✅ Large-scale environmental graphics / projection
- ✅ Data-visualization as aesthetic object (not for analysis)
- ✅ Tech company installations / branded experiences
- ✅ When project celebrates computation as creative collaborator
- ❌ NOT for static print work (anti-pattern — loses time dimension)
- ❌ NOT for legible information design
- ❌ NOT for quiet/intimate contexts (too spectacular)

## How to apply (tactical for Claude)
1. **Start from a dataset concept** — imagine: what is being visualized? Palette derives from source.
2. **Palette: highly saturated neon OR dataset-derived tones.** Choose Mode 1 (neon data), Mode 2 (architectural mono), or Mode 3 (organic data).
3. **Flowing point-cloud forms** — particles/dots clustered in dense formations, not hard shapes.
4. **Smooth gradients between states** — no hard edges, latent-interpolation feel.
5. **Architectural composition** — center-filling or edge-bleeding, scale to venue.
6. **Imply motion** even in static — visual momentum, streaks, flow direction.
7. **No typography-forward decisions** — metadata only, small and clinical.
8. **Reflective / refractive** quality — imagery feels like it moves through liquid/atmosphere.
9. **Symmetric when architectural** — projection onto regular buildings demands axial balance.
10. **NOT to use:** hard vector shapes, flat graphic-design palettes, figurative imagery, typographic hierarchy as focus, print-scale static composition without implied motion.
