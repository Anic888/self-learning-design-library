# Peter Saville — Style Profile

## Core Identity
**Cerebral minimalism via appropriation**. Borrowed classical imagery (Fantin-Latour roses, pulsar data, Renaissance paintings) recontextualized with zero contemporary decoration. Nothing shouts; everything implies. Design as intellectual puzzle — if you don't recognize the reference, you miss the meaning, but the surface is still beautiful.

## Palette
Saville varies dramatically per project — palette choice IS part of the reference being made. Tendencies:
- **Classical-painting palette**: muted oils (deep reds, dark greens, amber) — taken from source artworks
- **Near-black + bone white** — cover minimalism (Unknown Pleasures)
- **Systematic color-code** — New Order's color language (specific hexes tied to specific letters, see Power Corruption & Lies)
- **Muted off-whites, ghosts of color**
- **Factory industrial** — black + yellow-hazard, rare
- **Rule:** palette chosen from the referenced artifact, not from a "style palette". Research-driven.

## Typography
- **Minimal — often absent** on covers (Unknown Pleasures has NO type on front)
- When present: classical typefaces (Baskerville, Bodoni, Garamond) — referring to era/source
- **Bauhaus / constructivist sans** for certain eras (Factory catalog numbering)
- Ultra-clean typesetting — no distress, no overlap, no emotion
- **Small sizes common** — type doesn't compete with imagery
- **Centered or off-center discretely** — never chaotic
- Information hierarchy through whitespace + placement, not weight

## Composition & Grid
- **Appropriated image occupies majority** — type minimal around it
- **Centered or classical composition** — mirrors art-historical framing
- **Generous margins** — image is precious object
- **Flat color fields** when used — art as background
- **Systematic series thinking** — each record in a series relates to others
- **Minimal overlays** — image usually untouched, unless intentional over-paint

## Signature Moves
- **Appropriation** — use pre-existing image (classical painting, scientific diagram, photograph) as primary content
- **Nothing on front** — title/artist on back or not at all (Unknown Pleasures, Closer)
- **Color-coded meaning systems** — colors become language (New Order's alphabet on Power Corruption & Lies)
- **Classical-painting references** — Fantin-Latour, tomb photography, Italian Renaissance
- **Industrial/technical imagery** — floppy disk (Blue Monday), scientific diagrams
- **Ghosts of referenced artifacts** — muted colors taken from source painting
- **Serial numbering** — Factory's FAC-### catalog system became aesthetic
- **Postmodern irony** — sincere borrowing, but knowing; layered meaning
- **Spartan typography when used**

## Techniques / Tools
- Traditional paste-up era initially (1978-1990s)
- Later: standard Mac design tools (QuarkXPress → InDesign, Illustrator, Photoshop)
- Heavy use of high-resolution appropriated imagery (painting scans, scientific diagrams)
- Print production expertise — knows paper, ink, trim, binding

## Mood / Emotional Register
Contemplative, cool, intellectually generous but demanding. "I trust you to look this up." Not cold like Vignelli (function-first), not warm like Hara (sensory). **Scholarly-romantic**. Record collector aesthetic.

## When to apply
- ✅ Album / book covers wanting "thinking object" feel
- ✅ Cultural institution identity (museums, galleries)
- ✅ Fashion houses with heritage (Burberry, Selfridges)
- ✅ When project benefits from quotation / reference
- ✅ When quiet mystery is more powerful than loud assertion
- ❌ NOT for commercial / sales-driven (too coy)
- ❌ NOT for time-sensitive information (hides hierarchy)
- ❌ NOT for audiences who won't recognize references (joke falls flat)

## How to apply (tactical for Claude)
1. **Pick the reference first.** Before designing, identify the classical/scientific/historical artifact being quoted.
2. **Let the reference dominate.** Image at 60-80% of composition, minimal addition.
3. **Palette pulled from the reference.** Not applied from outside — extracted from source.
4. **Minimal or absent typography** — especially on cover/hero side. Type on back, colophon, metadata only.
5. **Classical typeface** when type is present — Baskerville/Garamond/Bodoni for literary feel; Bauhaus sans for constructivist feel. Match the era of the reference.
6. **Centered or classically-framed composition** — mirror how the source would be displayed in its original context.
7. **Generous margins** — treat the image as precious.
8. **Color-code systems if there's a series** — each piece in a series has a color logic connecting them.
9. **Don't decorate the reference.** No borders, no effects, no type overlaying image unless content-essential.
10. **NOT to use:** contemporary decorative elements, heavy typography, gradient effects, loud color, centered-ironic meme-like references. Saville's irony is sincere and subtle.
