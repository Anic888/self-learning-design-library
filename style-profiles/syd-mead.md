# Syd Mead — Style Profile

## Core Identity
Industrial-grade realistic futurism. Pre-digital concept art с painterly execution. Serious, believable, non-ironic futures. Создатель visual DNA современного sci-fi.

## Palette
- `#F4B860` / `#E89C4A` — warm amber / industrial orange (signature light, 15-25%)
- `#1A3B5C` / `#0F2744` — deep industrial blue (shadow/atmosphere, 20-30%)
- `#8B9094` / `#5F6368` — muted metallic greys (surfaces, 25-35%)
- `#FFFFFF` / `#F5F5F0` — warm white (key highlights, 5-10%)
- `#C43E1C` — deep red (accent для movement/danger, rare, 2-5%)
- **Тенденция:** Наземная палитра — не cyber-neon. Realistic lighting environment.

## Typography
- **Почти отсутствует в concept art.** Иногда industrial labels (tech callouts, model numbers)
- Когда есть: **technical sans** (Helvetica, Univers) или stencil military
- **НИКАКОГО эмотивного типографского языка** — только functional labels

## Composition & Grid
- **Cinematic framing** — 16:9 / wide composition даже в poster format
- **Single hero object** чаще всего (vehicle, building, figure)
- **Atmospheric perspective** — sense of huge scale через haze/fog
- **Rule of thirds соблюдается, но не догматично**
- **Foreground-midground-background** — 3 clear planes

## Signature Moves
- **Atmospheric haze** — distant elements fade в warm grey/blue
- **Reflective surfaces** — chrome, glass, wet pavement
- **Rain-soaked streets** (Blade Runner influence back to him)
- **Cinematic light beams** — rays, volumetric lighting
- **Functional futurism** — everything LOOKS like it works (unlike fantasy sci-fi)
- **Warm light / cold shadow** dichotomy (amber vs deep blue)
- **Human scale elements** для context размера

## Techniques / Tools
- **Gouache / acrylic на board** (pre-digital — это его эра)
- Высокая painting skill — realistic rendering
- Careful perspective drafting

## Mood / Emotional Register
Majestic, contemplative, optimistic-industrial. "Humans have built this, it's impressive and inhabited." Противоположность cyberpunk-cynicism.

## When to apply
- ✅ "Serious" sci-fi с industrial backdrop
- ✅ Concept art / vehicle / architecture visualization
- ✅ Когда нужна reference для atmospheric rendering
- ✅ Poster work где хочется gravity / weight
- ❌ НЕ подходит для neon-cyberpunk (это его "потомки" Josan/Signalnoise)
- ❌ НЕ для flat design / minimalist UI

## How to apply (tactical для Claude)
1. **Палитра: amber + deep blue + metallic grey.** Warm/cold dichotomy. Не neon.
2. **Композиция: one hero subject.** Вокруг — atmospheric context.
3. **Haze distance** — дальние элементы faded, warmer/bluer tone.
4. **Ground plane с reflections** (wet pavement / polished floor).
5. **Volumetric light** — видимые light rays / shafts.
6. **Human figures for scale** — всегда добавь 1-2 small figures для context.
7. **Typography: почти нет.** Минимальный label если нужен.
8. **НЕ использовать:** neon colors, flat shapes, grid-heavy composition, cyberpunk bloat. Mead не про это.
