# Ash Thorp — Style Profile

## Core Identity
Cinematic UI and sci-fi design через минимализм, thin linework, atmospheric depth. Ощущение "серьёзной" технологии — ни cyberpunk-camp, ни retro-nostalgia.

## Palette
- `#0E1419` / `#06090D` — deep near-black base (60-70%)
- `#F2F2F2` / `#E8E8E8` — бледно-белый (основной ink, 15-25%)
- `#FF6A00` / `#FF4500` — жжёный оранжевый accent (BR2049 influence, 5-10%)
- `#5AC8FA` / `#2A9FE3` — cold cyan (опционально, вместо оранжевого, 5-10%)
- **Тенденция:** monochromatic + ОДИН тёплый ИЛИ холодный accent. НЕ оба сразу.

## Typography
- **Headlines:** условные геометрические sans (Eurostile, Aktiv Grotesk Extended, GT America Mono). Почти always uppercase, wide tracking.
- **Body:** технический mono или neutral sans (Input Mono, Helvetica Now Mono)
- **Characteristics:** Низкая контрастность внутри шрифтов (thin / light веса доминируют), letter-spacing часто wide (100-300), text как data readout а не эмоциональный subject

## Composition & Grid
- **Сильно grid-driven.** Модульная сетка 12-24 columns.
- Много **negative space** — не заполняет всю площадь.
- Focal point обычно **слегка вне центра** или в нижней трети.
- Layered depth: 3-5 visual planes, разделённых только тонкими линиями
- Композиции часто **horizontal horizon-anchored** даже в вертикальных форматах

## Signature Moves
- **Thin glowing lines** (0.5-1px) как главный формообразующий элемент
- **HUD panels** — мелкие data-группы по углам и краям
- **Atmospheric perspective** — depth через opacity/blur а не через размер
- **Typographic data readouts** — time, coords, status tags как графический элемент
- **Quiet monochrome + one warm accent** почти всегда
- **No decorative gradients** — gradients только если они imply depth/atmosphere

## Techniques / Tools
- Cinema 4D + Redshift для 3D
- After Effects для motion
- Illustrator для layout/UI
- Photoshop для finishing

## Mood / Emotional Register
Контемплативный, режиссёрский, cold-tech-sublime. Не угрожающий но отчуждённый. Ближе к Kubrick чем к Nolan.

## When to apply
- ✅ Cinematic posters, title sequences
- ✅ UI-driven compositions
- ✅ "Serious" sci-fi / film-related
- ❌ НЕ подходит для party flyers, maximalist, playful work

## How to apply (tactical для Claude)
1. **Палитра: max 3 цвета.** Dark base + near-white + ОДИН accent (тёплый ИЛИ холодный).
2. **Используй ТОНКИЕ линии** (0.5-1px stroke) для всех декоративных элементов. НЕ solid shapes где можно strokes.
3. **Много breathing room.** Если content занимает >60% холста — переделывай.
4. **Типографика — condensed, wide-tracked, uppercase.** Никаких script, никаких heavy slab.
5. **Добавь minimum 2 группы HUD-readouts по углам** (time/coords/status)
6. **Horizon line** как composition anchor даже если это poster.
7. **НЕ использовать:** glitch effects, scanlines как main element, neon gradients, sunset colors. Это не его язык.
