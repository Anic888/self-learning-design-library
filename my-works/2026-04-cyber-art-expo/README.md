# 2026-04 — Cyber Art Expo Poster

> School assignment 7 — Adobe Illustrator poster (8.5×11 vertical).

## Brief
Создать 8.5×11 vertical poster в cyberpunk стиле. Requirements: readable colored type, pattern, limited palette, command of Illustrator tools (point type, area type, type on path, type mask, patterns).

## Concept
"NEO//GENESIS" — fictional Cyber Art Exhibition 2077. Blade Runner 2049 meets Ghost in the Shell minimalism с synthwave grid floor.

## Style references applied
- **Ash Thorp** — HUD corners, thin linework, horizon-anchored composition, quiet monochrome + one accent
- **James White / Signalnoise** — perspective grid floor, neon accents, sunset-adjacent palette
- **Josan Gonzalez** — pink/cyan/yellow neon palette (3-color cyberpunk)

## Palette (4 colors)
- `#0A0014` — deep indigo base
- `#FF006E` — hot magenta accent
- `#00F0FF` — electric cyan secondary
- `#FFFFFF` — white highlight

## Typography
- **Headlines:** Impact / Arial Black (fallback)
- **Details:** Courier New (mono для tech readouts)

## Assignment requirements demonstrated
| Requirement | Где в файле |
|---|---|
| Point type | Layer_MainTitle (NEO//GENESIS) |
| Area type | Layer_EventDetails (left/right columns) |
| Type on path | Layer_TypeOnPath (curved "CURATED SELECTIONS...") |
| Type mask | Layer_TypeMask ("2077" filled with iconPattern) |
| Simple pattern | Layer_Scanlines, Layer_DotGrid, iconPattern |
| Readable colored text | All typography has strong contrast vs dark bg |
| Limited palette | 4 colors |
| Layered composition | 16 named layers |

## What I learned / reflection
- SVG с правильной layer structure напрямую открывается в Illustrator как editable artwork
- Retrofuturism palette (pink + cyan + yellow) остаётся effective но needs restraint (used only pink + cyan + white here)
- RGB split headlines = effective glitch effect без filters
