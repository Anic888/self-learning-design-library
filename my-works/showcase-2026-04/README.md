# Showcase — April 2026: Legibility test across 4 styles

> The core claim of this library is that LLMs guided by distilled style profiles produce **legibly distinct** output — a viewer should be able to look at two pieces and say "this is Thorp, this is Carson." This folder is the proof.

Same underlying capability (Claude generating SVG). Same general prompt ("make a design"). Only the **referenced style profile** changes. If the profiles work, the outputs should be visually unmistakable from each other.

## Visual language comparison

| Dimension | Thorp (00) | Signalnoise (01) | Shinkawa (02) | Carson (03) |
|---|---|---|---|---|
| **Base color** | Near-black `#06090D` | Deep violet `#1E0B3A` | Bone white `#F5F0E0` | Newsprint `#F5F0E0` |
| **Accent** | Amber `#FF6A00` | Magenta + cyan + yellow (4-5 colors) | Oxblood red `#A33020` (one seal) | Oxblood red `#A33020` |
| **Typography** | Condensed light sans, 14px wide tracking | Italic Impact + chrome gradient | None dominant — one kanji + micro caption | 5 different fonts clashing, partial off-page |
| **Composition** | Horizon-anchored, 45% negative space, centered hero construct | Triangle frame + perspective grid + sunset sky | Single figure, 55% negative space as "paper" | NO grid, overlapping, tilted, broken |
| **Linework** | Hairline 0.3–0.8px | Gradients + neon bars | Wet brush simulation, varying stroke width | Xerox distressed, halftone |
| **Pattern** | Dot grid + tech grid (subtle) | Film grain | Paper noise | Halftone + paper noise |
| **Mood** | Meditative, cold-tech-sublime | Nostalgic optimism, romantic 80s | Martial restraint, reverent | Rebellious, intuitive, anti-corporate |
| **Hero move** | Concentric rings + amber horizon dot | Chrome italic type + sunset + grid | Gestural ink figure + red seal | Big tilted name + dingbats substitution |
| **What it rejects** | Neon / glitch / scanlines as decoration | Black backgrounds, minimalism | Color gradients, digital shading | Legibility, clean grid, balanced layout |

## Proving the claim

Look at any two pieces side by side. The differences are immediate:

- **00 vs 01:** Thorp = cold meditative monochrome. Signalnoise = warm maximalist gradient. Opposite emotional registers.
- **00 vs 02:** Thorp = digital vector precision. Shinkawa = physical brush simulation. Opposite execution philosophies.
- **00 vs 03:** Thorp = obsessive grid discipline. Carson = grid-breaking rebellion. Opposite relationships to structure.
- **01 vs 02:** Signalnoise = 5-color synthwave maximalism on violet. Shinkawa = 2-color ink monochrome on paper. Opposite palette philosophies.
- **01 vs 03:** Signalnoise = centered polished chrome. Carson = off-axis photocopied chaos. Opposite production values.
- **02 vs 03:** Shinkawa = silent eastern gesture. Carson = screaming western type. Opposite cultural approaches.

## How this was made

Each SVG was generated by Claude following the tactical rules in its corresponding style profile. Profiles contain:
- Exact hex palette assignments
- Typography specs (family, weight, tracking)
- Composition rules (grid / no grid, focal point tactics)
- Signature moves list
- Explicit **negative rules** ("do NOT use neon", "do NOT use balanced composition")

When Claude reads e.g. `style-profiles/david-carson.md` and sees `"NOT to use: clean sans-serif headline in center, balanced composition, modular grid"`, it deliberately breaks those defaults. That's how you get output that feels like Carson rather than generic "magazine design."

## Caveats / honest notes

- **These are SVG outputs.** Actual design production in tools like Illustrator/Figma would have manual refinements beyond what code-generation can do (custom kerning, real fonts, hand-drawn details).
- **Style "fidelity" is relative.** Compared to the actual designers, these are "recognizable interpretations," not reproductions. The library produces work that clearly **references** a style — it doesn't replace the designer.
- **The 4 pieces share a tool (SVG).** Different executions (3D render, ink on paper, letterpress print) would be more dramatic but require different tooling.
- **No ML training happened.** This is purely prompt-time retrieval. See `../../TRAINING.md` for discussion.

## What this demonstrates

1. **Legibility of distinction** — profiles produce recognizably different outputs (claim validated)
2. **Breadth of styles covered** — library handles genres as different as synthwave + sumi-e + chaos typography + cinematic UI
3. **Tactical rules work** — explicit "do / don't" instructions in profiles prevent generic defaults
4. **Consistency is achievable** — same tool (Claude + SVG), four very different aesthetic outputs
